{# Install on Ubuntu #}

2. Install on Ubuntu



Dingo is available on both the Ubuntu Software Center and Launchpad. If you prefer Launchpad then just open a terminal and use the following commands to add the official Dingo Team PPA (which contains the latest stable release): 

      $ sudo add-apt-repository ppa:dingo-team/dingo-stable
      $ sudo apt-get update
      $ sudo apt-get install dingo
{# Install on Fedora #}

2. Install on Fedora

On Fedora Dingo can be installed via yum. Just open your terminal and input the following commands: 

      $ yum install dingo
{# Install on Other Distros #}

2. Install on Other Distros

Dingo binary is currently only available on Fedora and Ubuntu-based systems. To install Dingo for other systems please following these steps:

  1. Make sure you have installed all the necessary dependencies
  2. Download the latest stable Dingo source release (see the section below)
  3. cd into the project's directory and type the following command:
      $ cd /path/to/dingo/directory
      $ ./configure
      $ make
      $ make install
