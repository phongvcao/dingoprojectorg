Report Bugs

For easy integration with the development process, Dingo uses the default Github issues tracker for bug reports. The tracker is very intuitive and easy to use. Learn more about how to use Github issues tracker.


Before filing a bug, please go visit our list of recent duplicates.


We recommend you to take a look at our wiki article on Github on How to write a good bugs report.


When you're ready, file your bug report.


If your bug involves unexpected behaviors or an interface error please attach a screenshots with some notes explaining what happened.


Please DO NOT send bug reports to the maintainers' emails or the mailing-list. Only the bugs that are properly entered into our Github issues tracker are fixed.
