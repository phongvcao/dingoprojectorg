Translate Dingo

Dingo is shipped supporting two languages by default: English and Vietnamese. However, there are dozens of languages out there that have not been integrated into Dingo. Just by translating some text you can help Dingo expand its user base.


Before translating Dingo please check out the translations currently available for the latest release of Dingo and their progresses on our Github repository. Then please read translating GNOME applications.


If possible, you should contact the existing translators or translation team of the language you want to work on (their email addresses are noted in the po files) to avoid duplicates and conflicts during the translation process.


To translate Dingo, you can download the po files or get the latest development version of Dingo.


After completing the translation, if you know how to use Git and Github please do a pull request on our Github repository. Otherwise just contact your translation team leaders and send them the patch.
