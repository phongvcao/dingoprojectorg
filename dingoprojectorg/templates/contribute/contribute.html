Dingo is free and open-source just like many other Linux software, and you are more than welcome to help us make it better regardless of how geeky you are: developers, designers, translators, normal users, etc. all-in!

{# Report Bugs #}

Report Bugs

If you find bugs in Dingo just report to us using our Github bugs tracker. However, before filing the bugs please check the bugs list to see if there is any duplicate since the problem might have already been resolved. More about filing bugs.


Suggestions for new features and enhancement requests should be sent to our mailing-list. Please check our mailing list archives for duplicates before filing one.

{# Translate Dingo #}

Translate Dingo

Dingo is shipped supporting two languages by default: English and Vietnamese. However, there are dozens of languages out there that have not been integrated into Dingo. Just by translating some text you can help increase Dingo's popularity. Learn more about translating.

{# Design Icons #}

Design Icons

User interface is an important part of a software, and nice-looking icons are necessary in constructing friendly user interface. Dingo was created using mainly the default system icons plus other software's artworks, which was really not enough. That's why we need you, designers! Learn more about designing icons.

{# Write Code #}

Write Code

We welcome everybody to join our development process, regardless your programming level. The author of Dingo was only a college student when he set things up and it is likely that you are better than him. Learn more about writing code for Dingo.
